Covid-19 Visualizer 🦠

A map that displays the global cases and deaths in every country and province.

A React app using Mapbox API to generate interactive app as well as updated information from John Hopkins University taken from this API (https://corona.lmao.ninja/v2/jhucsse). Each country and province affected will have a marker, whose colour and size depends on the severity of the disease there. The user can hover over it to see more information. Deployed using Netlify.
